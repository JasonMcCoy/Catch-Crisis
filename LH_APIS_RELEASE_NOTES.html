After Update Compilation Help

Because of changes in the used libraries files & code structure, user action is required to make an old project compile.
A. Change your Xcode project "Header Search Paths"

1. Select you [ProjectName] + Box2d - iOS target
2. Navigate to Build Settings
3. Search for "Header Search Paths"
4. Double click and add the following paths in both Debug & Release
  • "$(SRCROOT)/LevelHelper2-API/LevelHelper2-API" - "recursive"
  • "$(SRCROOT)/Box2D" - "recursive"
5. Go to LHSceneDemo.mm if still present in your project and change:
#import "LevelHelper2-API/LHUtils.h"
to
#import "LHUtils.h"
